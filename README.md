# Bonjour avec Phoebe 🎌

Platform belajar Bahasa Prancis untuk pengguna berbahasa Indonesia — dibangun dengan Next.js 16 (App Router), TypeScript, Tailwind CSS, Supabase, dan Google Gemini API.

## Getting Started

```bash
npm install
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000) di browser.

## Environment Variables

Salin `.env.example` ke `.env.local` dan isi:

```
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
GEMINI_API_KEY=
```

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Supabase (Auth + Database)
- Google Gemini API (`gemini-1.5-flash`)
- Deploy: Vercel
